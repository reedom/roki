# Project Guidance

## Project Memory
Project memory keeps persistent guidance (component docs) so the agent honors your standards each run.

- Use local `AGENTS.md` files for feature or library context (e.g. `src/lib/payments/AGENTS.md`): describe domain assumptions, API contracts, or testing conventions specific to that folder. They auto-load when working in the matching path.

## Development Guidelines
- Think in English, generate responses in English.
- Follow user instructions precisely. Within that scope act autonomously: gather context and complete the requested work end-to-end, asking only when essential information is missing or critically ambiguous.
